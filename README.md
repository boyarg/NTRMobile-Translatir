# NTR Mobile - Portable Auto-Translator

especially for NosyFellow https://www.patreon.com/cw/NosyFellow

Offline, no-download-needed translator installer for **NTR Mobile** using BepInEx 6 + XUnity AutoTranslator 5.6.1.

All required files are included in the `files/` folder. **No internet connection required** for installation (translation itself still needs internet).

## Quick Start

1. Double-click **`install.bat`**
2. Enter the full path to your NTR Mobile game folder  
   (e.g. `D:\Games\NTRMobileV0.23.1`)
3. Select the **source language** (the language the game is currently in)
4. Select the **target language** (the language you want to translate to)
5. Select a **translation service** (GoogleTranslateV2 is recommended)
6. Press any key to begin installation
7. Launch the game!

## What Gets Installed

The script copies the following from the `files/` folder into your game directory (~74 MB total):

| File/Folder | Purpose |
|-------------|---------|
| `winhttp.dll` | BepInEx loader (Doorstop) |
| `.doorstop_version` | Doorstop version marker |
| `baselib.dll` | Base library |
| `doorstop_config.ini` | Doorstop configuration |
| `dotnet/` (.NET 6 runtime) | CoreCLR runtime required by BepInEx IL2CPP |
| `BepInEx/core/*` | BepInEx 6 core runtime |
| `BepInEx/plugins/XUnity.AutoTranslator/*` | Translation plugin |
| `BepInEx/plugins/XUnity.ResourceRedirector/*` | Resource redirector plugin |

Then it generates a configuration file (`AutoTranslatorConfig.ini`) with your language and service choices.

## After Installation

1. **Launch the game.** The first launch will be slow because BepInEx generates interop assemblies (this is normal, just wait).
2. Translation starts automatically. Text is translated on-the-fly via the selected service.
3. All translations are cached in `_AutoGeneratedTranslations.txt` and reused on later launches.

## File Locations

After installation, your game folder will contain:

```
GameFolder/
├── winhttp.dll                         ← BepInEx loader
├── .doorstop_version
├── baselib.dll
├── doorstop_config.ini
└── BepInEx/
    ├── config/
    │   └── AutoTranslatorConfig.ini    ← Main configuration
    ├── core/                            ← BepInEx runtime
    ├── plugins/
    │   ├── XUnity.AutoTranslator/       ← Translation plugin
    │   └── XUnity.ResourceRedirector/   ← Resource plugin
    └── Translation/
        └── {lang}/
            ├── Text/
            │   ├── _AutoGeneratedTranslations.txt  ← Cached translations
            │   ├── _Substitutions.txt              ← Manual overrides
            │   ├── _Preprocessors.txt
            │   └── _Postprocessors.txt
            ├── Texture/
            └── RedirectedResources/
```

## Manual Translations

Edit `_Substitutions.txt` to add your own translations:

```
Original English text=Your translated text
```

Each line defines a direct substitution. These take priority over automatic translation.

## Troubleshooting

### Game doesn't translate anything

1. Check `BepInEx\LogOutput.log` for errors
2. Make sure `FromLanguage` matches the actual game language
3. Batching is disabled by default to avoid errors. If translations are very slow, you can try enabling `EnableBatching=True` in `AutoTranslatorConfig.ini`

### "Batch operation received incorrect number of translations" error

This is already fixed in this installer (batching is disabled). If you re-enabled it and see this error, set `EnableBatching=False` in `AutoTranslatorConfig.ini`.

### "Could not hook TMP_Text_SetCharArray" warning

Known issue with Unity 6. Some TextMeshPro text may not translate. Try enabling `EnableTextMesh=True` in the config.

### First launch is very slow

Normal behavior. BepInEx generates interop assemblies on first launch (~1-2 minutes). Subsequent launches are fast.

## Uninstallation

To remove the translation framework and restore the original game:

1. Delete from your game folder:
   - `BepInEx\` folder
   - `dotnet\` folder
   - `winhttp.dll`
   - `.doorstop_version`
   - `baselib.dll`
   - `doorstop_config.ini`

## Distribution

To share this translator with others, simply zip the entire `portable_translate` folder (including `files/` and `install.bat`). Recipients just need to:
1. Unzip
2. Run `install.bat`
3. Enter their game path and language preferences
4. Play!
